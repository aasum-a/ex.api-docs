# Dokumentasi Proyek: Landing Page Mahardika Sablon

Selamat datang di dokumentasi *development* proyek Landing Page & Admin Panel Mahardika Sablon.

Dokumen ini mencatat arsitektur teknis, keputusan desain, dan (yang paling penting) log *troubleshooting* besar yang dihadapi selama proses *development*. Tujuannya adalah untuk menyediakan konteks penuh agar *development* dapat dilanjutkan secara efisien.

**Tujuan Awal Proyek:**
Membuat *admin panel* (CMS) kustom menggunakan Laravel 12 dan Filament untuk mengelola konten dinamis (Portfolio, Testimoni, Klien, Pengaturan Halaman) dari sebuah *landing page* statis.
